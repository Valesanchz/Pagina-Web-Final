Bienvedido/as a mi página acerca de mí

El aprendizaje que obtuve en esta Unidad acerca de Html y Css me ayudó a conocer algunas de las herramientas que se necesitan para crear una página web. Esto me ayudó a conocer más sobre el internet, a como usar correctamente HTML y CSS, también aprendí a utilizar mi creatividad en los entornos digitales. Pude diseñar y crear páginas web con elementos como el fondo, textos en varios formatos, la alineación, muchos colores, imágenes, tablas, y muchas cosas más.

Conocí sobre HTML y CSS que son herramientas complementarias e indispensables que tuve que poner en práctica para la realización de mis actividades, con diferentes diseños y estética.
